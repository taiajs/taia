## create app

Let's find a place to create an empty directory

```shell
$ mkdir myapp && cd myapp
```

Create projects through official tools

```shell
$ npx create-taia@latest
# or
$ yarn create taia
# or
$ pnpm dlx create-taia@latest
```

## start a project

```shell
$ npm start
$ i
# or
$ a
```

## build

```shell
$ npm run build
```
